# India-inflation-data-analysis
Exploratory data analysis of India inflation data using Python and Pandas

# India Inflation Data Analysis

## Objective
To analyze India’s inflation trends using real government data and derive
economic insights through exploratory data analysis.

## Dataset
- Source: Government of India / CPI-based inflation data
- Frequency: Monthly
- Period: 2018–19 to 2022–23

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Analysis
- Data cleaning and restructuring (wide to long format)
- Year-wise average inflation
- Pre-COVID vs Post-COVID comparison
- Trend visualization

## Key Insights
- Inflation shows significant volatility across years
- Post-COVID period exhibits higher average inflation
- Monthly data helps identify short-term price pressures
